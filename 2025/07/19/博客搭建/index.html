一 安装node.js等

安装node.js,git,github注册。

二 安装hexo

先创建博客目录，例如myblog.然后安装hexo：打开命令行工具，使用cd命令进入博客目录，运行以下命令npm install -g hexo-cli进行安装。安装完以后，初始化博客。使用hexo init命令。

三 配置hexo

具体查看官网[hexo](https://hexo.io/themes/)

四 本地预览博客

在博客目录下的命令行中运行hexo server命令，启动成功后，在浏览器中输入本地访问地址可看到博客预览效果。

五 将博客部署到github

创建github仓库后安装 hexo - deployer - git 插件，部署配置信息，最后部署博客
